## 四、核心算法设计

### 4.1 标尺识别与尺寸校准（所有测量的基础）

```python
# 伪代码示例：标尺识别与像素-毫米转换
def calibrate_scale(image):
    """
    识别照片中的5mm参考标尺 → 计算像素/mm转换比例
    """
    # 1. 在图像中检测已知尺寸的标尺标记（两个圆点间距5mm）
    markers = detect_fiducial_markers(image, pattern='double_dot')
    # 2. 计算像素距离
    pixel_distance = euclidean_distance(markers[0], markers[1])
    # 3. 计算转换比例
    scale = 5.0 / pixel_distance  # mm/pixel
    return scale
```

### 4.2 洞形外形轮廓评分（20分）— 模板匹配法

```
原理：将学生制备的洞形轮廓与标准Ⅱ类洞模板进行相似度比较

┌─────────────────────────────────────────────────────┐
│                  外形轮廓评分                         │
│                                                     │
│   标准模板                  学生作品                   │
│   ┌─────────┐             ┌─────────┐               │
│   │  ╱‾‾‾╲  │             │  ╱‾‾‾‾╲ │               │
│   │ ╱ 鸠尾 ╲│             │ ╱  鸠尾 ╲│               │
│   │╱  峡部  ╲│    vs      │╱  峡部  ╲│               │
│   │   邻面  │             │   邻面  │               │
│   │   盒形  │             │   盒形  │               │
│   └─────────┘             └─────────┘               │
│                                                     │
│   评分算法：                                         │
│   1. 边缘检测(Canny) → 学生轮廓S                     │
│   2. 刚性配准 → 将标准模板T对齐到学生轮廓              │
│   3. 计算Dice相似系数 = 2|S∩T|/(|S|+|T|)            │
│   4. 外形分 = 20 × Dice系数                         │
│                                                     │
│   扣分规则：                                         │
│   · 鸠尾过大/过小：-3/级                            │
│   · 邻面盒形不完整：-5                              │
│   · 外形线不连续：-2                                │
└─────────────────────────────────────────────────────┘
```

### 4.3 鸠尾峡评分（15分）— 比例测量法

```
测量指标：
┌────────────────────────────────────────────┐
│                鸠尾峡测量                   │
│                           颊侧             │
│   ┌─────────────────────────┐             │
│   │         鸠尾             │             │
│   │      ┌───────┐          │             │
│   │      │ 峡部  │← 峡宽W   │             │
│   │      └───────┘          │             │
│   │           │              │             │
│   │           │ 邻面盒形     │             │
│   │           │              │             │
│   └─────────────────────────┘             │
│               舌侧                          │
│   ←———— 颊舌尖总宽度B ————→               │
│                                            │
│   评分公式：                                │
│   · 理想比值 R_ideal = W/B = 1/3           │
│   · 实际比值 R_actual = AI测量值            │
│   · 峡宽分 = 15 × e^(-50×(R_actual-0.33)²)│
│     （高斯函数：越接近1/3得分越高）           │
│                                            │
│   · 峡部位置分 = 检测峡部是否位于邻面入口处    │
│     （位置偏移>1mm → 扣3分）                │
└────────────────────────────────────────────┘
```

### 4.4 洞深评分（10分）— 色差分析法

```
原理：牙釉质呈乳白色/半透明，牙本质呈淡黄色，
      通过色彩空间分析判断是否到达釉牙本质界下0.5mm

┌─────────────────────────────────────────────┐
│              色差深度分析                     │
│                                             │
│  1. 颜色空间转换（RGB → LAB）                 │
│     · L通道：亮度（区分釉质/牙本质）           │
│     · B通道：黄蓝色（牙本质偏黄，B值低）        │
│                                             │
│  2. 深度映射：                               │
│     · 建立"色差值 → 深度"的标定曲线            │
│     · 釉质层：L>85, B>5                      │
│     · 釉牙本质界：L=75-85, B=2-5              │
│     · 牙本质浅层(0.5mm)：L=65-75, B=-2~2 ←目标│
│     · 牙本质深层(>1mm)：L<65, B<-2 ←过深扣分  │
│                                             │
│  3. 评分：                                  │
│     · 每个像素点计算"深度偏差"                 │
│     · 洞深分 = 10 × (达标像素占比)             │
│     · 穿髓检测：B值异常低区域 → 警告"疑似穿髓"  │
│                                             │
│  校准方式：在标准离体牙上制备已知深度(0.5mm)    │
│           的参考洞形 → 建立色差-深度对应表      │
└─────────────────────────────────────────────┘
```

### 4.5 底平壁直评分（15分）— 平面拟合+角度测量

```
┌─────────────────────────────────────────────┐
│       底平壁直测量（需颌面正位+45°斜位）       │
│                                             │
│  髓壁平面度（8分）：                         │
│  ┌─────────────────────────────┐            │
│  │  从45°斜位图像中提取髓壁区域  │            │
│  │  对髓壁区域内所有像素点进行     │            │
│  │  最小二乘法平面拟合            │            │
│  │  计算每个点到拟合平面的距离残差  │            │
│  │                              │            │
│  │  平面度分 = 8 × e^(-λ×RMSE)  │            │
│  │  RMSE(均方根误差)越小分越高   │            │
│  └─────────────────────────────┘            │
│                                             │
│  侧壁垂直度（7分）：                         │
│  ┌─────────────────────────────┐            │
│  │  提取侧壁轮廓线                │            │
│  │  计算侧壁与髓壁的夹角          │            │
│  │  · 理想角度：90°±5°          │            │
│  │  · 角度偏差>15°：明显扣分     │            │
│  │                              │            │
│  │  垂直度分 = 7 × avg(cosθ_i)  │            │
│  │  (θ_i为各测量点的角度偏差)     │            │
│  └─────────────────────────────┘            │
└─────────────────────────────────────────────┘
```

### 延伸：五类洞全系列覆盖

| 洞型 | AI评分特点 | 独有指标 |
|:--:|------|------|
| Ⅰ类洞 | 颌面点隙裂沟，评分相对简单 | 外形对称性、点隙覆盖完整度 |
| **Ⅱ类洞** | **邻面+颌面复合，最复杂** | **鸠尾峡比例、邻面盒形、邻牙保护** |
| Ⅲ类洞 | 前牙邻面，美观要求高 | 唇面保存率、切缘保护 |
| Ⅳ类洞 | 前牙邻面+切角，最考验美观 | 切角恢复形态、中线对称 |
| Ⅴ类洞 | 牙颈部，肾形外形 | 龈壁深度、肾形对称性 |

→ Ⅱ类洞是最佳切入点（复杂性最高），跑通后扩展到全系列。

## 八、说课PPT中的展示建议

在说课PPT中展示AI辅助评分时，建议用**3张图**讲清：

1. **"拍照→AI分析→评分报告"流程图**（本文第三章）
2. **一张学生评分报告样例**（本文第五章，拿一个真实的Ⅱ类洞照片模拟生成）
3. **一张教师端的全班薄弱点分析图**（展示"数据驱动教学改进"的闭环）

口头讲解要点：
> "学生做完二类洞，用手机拍5张标准角度的照片上传系统。22秒后，AI会从12个维度给出量化评分——不是'感觉做得不错'，而是'鸠尾峡宽度0.42倍，偏宽，扣2分'这样的精确反馈。教师端会看到全班59人的薄弱点分布，如果42%的学生在髓壁平面度上失分严重，下次课就重点训练分层预备法。这就是从'经验驱动'到'数据驱动'的教学转变。"
